# Tenant Form for home.ht

## Tech

It is a React application.

To clone this repo: 

`$ git clone https://github.com/Tina-Trillian/home-quick-exercise`

To start the server-api and the react-app:

`$ cd home-quick-exercise`  
`$ npm install`  
`$ npm run dev` 

Server runs on PORT 1234
